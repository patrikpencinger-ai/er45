What this is

archive.er45.com is the first subdomain behind the ER45 memorial — the tagline “the infrastructure continues in the subdomains” made real. It's the actual content of ER45.com, the Croatian electronic-music portal (techno · house · drum'n'bass) that ran roughly 1999–2003, recovered from the original database.

Where the data comes from

Everything here is decoded straight from the original MySQL er45 database (MySQL-Front dump, server 3.23) and the Access stores the live ASP site ran on:

  • Chronology — the chrono table (the party calendar that was the backbone of the site), with venue names resolved against the lokacije table (510 venues).
  • Articles — the clanci table: DJ interviews, artist bios, party reports and gear pieces.
  • Flyers — the original event flyers from the Flyeri archive, dated from their filenames.

Note on the text

The source data was Windows-1250 encoded, and the article store had its Croatian letters double-mangled (č stored as è, ć as æ). Both were reversed during extraction, so the diacritics here — č, ć, š, ž, đ — are the originals, not approximations.

Provenance & privacy

This archive is built only from public site content — event listings, editorial and flyers. Personal data from the old database (logins, mailing lists, contest entrants, contacts) is deliberately excluded.
